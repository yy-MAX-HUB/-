## 核心功能

### 评论实时监控

- 10 秒级评论抓取，比人工刷新快 100 倍
- 同时监控 **300+** 视频，覆盖全行业热门内容
- 关键词智能筛选 + IP 地区过滤 + 黑名单自动拉黑
- 钉钉 / 企微通知推送，手机实时接收新评论

### 自动私信截流

- 新评论触发后自动向目标用户发送私信
- 支持多账号轮换发送，降低风控风险
- 话术模板随机轮换，每条私信都不一样
- 自动关注 + 自动点赞，提升私信到达率
- 单日发送上限、间隔频率全可配置

### AI 智能选品

- 输入业务目标（如"新疆旅游获客"），AI 自动生成搜索关键词
- 批量搜索抖音视频，按评论活跃度 / 互动率 / 时效性自动评分
- 一键将高分视频加入监控列表，省去人工选品时间

### 视频管理

- 一键添加 / 批量导入视频链接
- 实时显示每个视频的今日新增评论数和总评论数
- 无新评论视频一键清理，保持监控列表高效
- 点击输入框自动粘贴剪贴板内容，操作更快

### 在线售后客服

- 页面右下角"售后"按钮，点击即开聊天窗
- 支持文字 + 图片发送，沟通无障碍
- 消息加密传输，隐私安全有保障

## 技术架构

```
用户浏览器  ←→  FastAPI Web 后端  ←→   评论抓取引擎
                    ↕                         ↕
              Vue 3 前端               抖音评论数据
              WebSocket 实时推送        私信发送接口
              自动更新服务              多账号管理
```

- **前端**：Vue 3 + Tailwind CSS + ECharts，暗色/亮色主题自适应
- **后端**：Python FastAPI + WebSocket 实时通信
- **监控引擎**：独立评论抓取服务，高并发低延迟
- **私信系统**：多账号轮换 + Playwright 自动化
- **部署**：双击 exe 一键启动，自动打开浏览器，零配置
